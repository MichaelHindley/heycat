Show the current in-progress issue status:

1. Run `bun .claude/skills/agile/agile.ts list --stage 3-in-progress` to find the issue
2. Run `bun .claude/skills/agile/agile.ts work <issue-name>` to show:
   - Spec completion progress
   - Technical guidance status
   - Definition of Done progress
   - Recommended next actions
